# Plexmatch Generator
This is a free `.plexmatch` file generator for Plex.

You can learn more on <a href="https://support.plex.tv/articles/plexmatch/" target="_blank">Plex official documentation</a> about `.plexmatch` files.

Tool is available [adriendeval.github.io/plexmatch-generator](https://adriendeval.github.io/plexmatch-generator/).
